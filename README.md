# DJ Democracy

## what even?

DJ Democracy is the democratic solution to what plays next at your party.

## okay, but how?

Weighted average scores for songs added to a queue determine the order in which the queue is played.

### if you're in need of step wise instructions

1. Create a room and share the link with your gang
2. Everybody in the room can add to the queue
3. Songs rise up in the queue based on a 1-5 score that everyone else in the room gives to each song
